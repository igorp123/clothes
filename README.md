# Clothes choice
## Что надеть? 

Учебная программа, которая рекомендует, как одеться сегодня по погоде.

Программа написана на **Ruby v 2.4.4**.
***

Запуск программы
``` ruby
  ruby main.rb
```

***
Файлы наборов вещей находятся в `\data`

Формат файлов
```
<Название одежды>
<Тип>
<Диапазон температур>
```
